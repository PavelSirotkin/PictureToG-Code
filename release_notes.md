# Release Notes — PictureToG-Code v0.9.1

**Дата:** 12 апреля 2026 г.

---

## Краткое описание

PictureToG-Code v0.9.1 — обновление, добавляющее алгоритмы сглаживания контуров, равномерный ресемплинг и HTML-версию приложения для работы в браузере.

---

## Новые возможности

### Сглаживание контуров Chaikin

- Добавлен алгоритм Chaikin для сглаживания углов и неровностей в контурах
- Позволяет получить более плавный G-код без резких переходов
- Уменьшает износ инструмента и улучшает качество обработки

### Ресемплинг по длине дуги

- Равномерное распределение точек вдоль контура
- Обеспечивает стабильное качество G-кода независимо от исходного разрешения изображения
- Устраняет неравномерные участки с избытком или недостатком точек

### HTML-версия приложения

- Полноценный веб-интерфейс для работы в браузере
- Возможность запуска без установки Python
- Все основные функции доступны в браузере (папка `Html/`)

---

## Улучшения

- Добавлены выходные `.nc` файлы в папку `Samples/` для тестирования и ознакомления
- Обновлена документация в README.md

---

## Исправления

- Исправлены проблемы с обработкой контуров при сложной геометрии
- Улучшена стабильность генерации G-кода

---

## Известные особенности

- HTML-версия может требовать современный браузер (Chrome, Firefox, Edge)
- Для работы с кириллическими путями при запуске из исходного кода требуется Python 3.8+

## Установка

### Из исходного кода

```bash
git clone https://github.com/PavelSirotkin/PictureToG-Code.git
cd PictureToG-Code
pip install -r requirements.txt
python main.py
```

### Из .exe файла

- Запустите скачанный `.exe` файл
- Никакой дополнительной установки не требуется
