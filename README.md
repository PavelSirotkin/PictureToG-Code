## Описание

**PictureToG-Code** — это приложение для конвертации растровых изображений в G-код, предназначенное для использования с фрезерными станками с ЧПУ. Программа поддерживает два режима работы: контурную обработку и создание 2.5D рельефа.

## Возможности

### Режимы обработки

- **Контур** — извлечение контуров из изображения и генерация G-кода по контурам
  - Бинаризация изображения с настраиваемым порогом
  - Размытие для сглаживания шумов
  - Упрощение контуров (алгоритм Рамера-Дугласа-Пекера)
  - Офсет контуров с учётом диаметра фрезы (требуется Shapely)
  - Мостики (bridges) для предотвращения смещения деталей
  - Многопроходная обработка

- **Рельеф** — создание 2.5D рельефа на основе яркости пикселей
  - Тёмные участки = глубже, светлые = выше
  - Две стратегии: Зигзаг и Однонаправленный
  - Настраиваемое перекрытие дорожек
  - Отдельная подача для врезания

### Параметрические шаблоны

- Брелок прямоугольный
- Брелок круглый
- Табличка с рамкой
- Звезда
- Сердце

### Дополнительные функции

- Визуальное превью контуров и карты высот
- Живое превью бинаризации при настройке порога
- Автосохранение настроек между запусками
- Подсветка синтаксиса G-кода
- Оценка времени обработки и расстояний
- Масштабирование с сохранением пропорций
- Тёмная тема интерфейса

## Установка

### Требования

- Python 3.8 или выше
- Windows / Linux / macOS

### Шаги установки

1. **Клонируйте репозиторий или распакуйте архив:**
   ```bash
   cd PictureToG-Code
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **(Опционально) Установите Shapely для улучшенных офсетов:**
   ```bash
   pip install shapely
   ```
   Без Shapely программа будет работать, но офсет контуров будет отключён.

## Использование

### Запуск приложения

```bash
python main.py
```

### Основные шаги

1. **Откройте изображение:**
   - Нажмите кнопку «Открыть изображение...»
   - Поддерживаемые форматы: PNG, JPG, JPEG, BMP, TIFF

2. **Выберите режим:**
   - **Контур** — для вырезания контуров
   - **Рельеф** — для создания 3D рельефа

3. **Настройте параметры:**

   #### Общие параметры
   | Параметр | Описание | По умолчанию |
   |----------|----------|--------------|
   | Диаметр фрезы | Диаметр инструмента (мм) | 3.175 |
   | Подача | Скорость рабочей подачи (мм/мин) | 800 |
   | Глубина | Конечная глубина обработки (мм) | 2.0 |
   | Число проходов | Количество проходов | 1 |
   | Safe Z | Безопасная высота (мм) | 5.0 |
   | Скорость шпинделя | Скорость вращения шпинделя (об/мин) | 15000 |

   #### Параметры контура
   | Параметр | Описание | По умолчанию |
   |----------|----------|--------------|
   | Упрощение | Степень упрощения контуров (мм, 0 = нет) | 0 |
   | Порог бинаризации | Порог разделения (0-255) | 128 |
   | Размытие | Размер ядра размытия (px) | 3 |
   | Мин. площадь | Минимальная площадь контура (px²) | 10 |
   | Аппроксимация | Коэффициент аппроксимации | 0.001 |
   | Инвертировать | Тёмное = контур | Вкл |
   | Мостики | Включить мостики | Выкл |
   | Размер мостика | Длина мостика (мм) | 3.0 |
   | Кол-во мостиков | Количество мостиков на контур | 2 |

   #### Параметры рельефа
   | Параметр | Описание | По умолчанию |
   |----------|----------|--------------|
   | Стратегия | Зигзаг / Однонаправленный | Зигзаг |
   | Перекрытие | Перекрытие дорожек (% диаметра) | 40 |
   | Подача врезания | Скорость при врезании (мм/мин) | 300 |
   | Размытие | Размытие карты высот (px) | 5 |

4. **Настройте размер вывода:**
   - Укажите ширину (W) и высоту (H) в мм
   - Или оставьте пустым (1 пиксель = 1 мм)
   - Кнопка замка включает/выключает сохранение пропорций

5. **Сгенерируйте G-код:**
   - Нажмите кнопку «Сгенерировать G-Code»
   - Проверьте превью и статистику
   - При необходимости скорректируйте параметры

6. **Сохраните результат:**
   - Нажмите «Сохранить .nc...»
   - Поддерживаемые расширения: .nc, .gcode, .tap

## Примеры

В папке `Samples/` находятся тестовые изображения. Используйте их для ознакомления с программой:

| Файл | Описание |
|------|----------|
| `picture1.jpg` – `picture5.jpg` | Различные изображения для тестирования |
| `Это длинное имя рисунка на кириллице.jpeg` | Тест кириллицы в имени файла |

### Контурная обработка

1. Откройте чёрно-белое изображение (или цветное — будет преобразовано)
2. Убедитесь, что выбран режим «Контур»
3. Настройте порог бинаризации так, чтобы нужные области были белыми, а фон — чёрным
4. При необходимости включите мостики
5. Нажмите «Сгенерировать G-Code»

### Создание рельефа

1. Откройте изображение с градациями серого
2. Переключитесь в режим «Рельеф»
3. Настройте размытие для сглаживания
4. Укажите желаемый размер вывода (мм)
5. Нажмите «Сгенерировать G-Code»

### Использование шаблонов

1. Выберите шаблон из выпадающего списка
2. При необходимости укажите размер
3. Настройте параметры обработки
4. Нажмите «Сгенерировать G-Code»

## Формат G-кода

Программа генерирует стандартный G-код с поддержкой:

- `G0` — быстрое перемещение
- `G1` — линейная интерполяция (рабочий ход)
- `G21` — режим мм
- `G90` — абсолютные координаты
- `G17` — плоскость XY
- `M03` — включение шпинделя
- `S` — скорость вращения шпинделя (об/мин)
- `M05` — выключение шпинделя
- `M30` — конец программы

## Настройки

Настройки автоматически сохраняются в файл `settings.json` при изменении параметров.

### Расположение файла настроек

- **При запуске из исходного кода:** рядом с корневым файлом скрипта
- **При запуске из .exe (PyInstaller):** рядом с исполняемым файлом

### Сброс настроек

Удалите файл `settings.json` — при следующем запуске будут использованы значения по умолчанию.

## Устранение неполадок

### Не загружается изображение

- Проверьте формат файла (поддерживаются: PNG, JPG, JPEG, BMP, TIFF)
- Убедитесь, что файл не повреждён

### Ошибка при генерации G-кода

- Проверьте, что все числовые параметры заполнены корректно
- Убедитесь, что глубина и диаметр фрезы больше нуля

### Офсет контуров не работает

- Установите библиотеку Shapely: `pip install shapely`
- Без Shapely контуры обрабатываются без компенсации диаметра фрезы

### Настройки не сохраняются

- Проверьте права доступа к папке с программой
- При запуске из Program Files могут потребоваться права администратора
